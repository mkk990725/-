# 学习失败经验技能

## 目标

对已结束比赛进行赛后复盘，让智能体从预测失败中更新分析纪律。复盘不是为错误找理由，而是找出下一次应该补充、降低权重或摒弃的信息。

## 复盘步骤

1. 对比预测与真实情况：上半场走势、全场走势、关键进球/失误、教练调整、换人、体能变化、市场预期和娱乐比分/半全场。
2. 给预测打分：0-100 分。只奖励赛前可验证的判断，不奖励事后解释。信息源不足却给出强结论要扣分。
3. 找失败原因：区分信息缺失、信息源误导、模型推理错误、权重错误、不可预测事件。
4. 调整 skill：说明哪条分析纪律需要加强、删除或降权。
5. 调整信息源：列出下一次必须补充的信息，和应该摒弃的噪声来源。

## 输出要求

- 输出“预测得分”“预测正确点”“预测失败点”“需要新增的信息”“需要摒弃的信息”“skill 调整建议”。
- 不要用单场特例泛化成普遍规律。只有多个样本反复出现的模式，才允许写入核心经验。
- 对烟雾弹信息要记录来源、传播路径和被证伪原因。
